# Data Structures Practice (C++)

This repository contains **11 C++ implementations** of core Data Structures and Algorithms.
These were developed as part of **lab assignments and personal practice**.

The goal here is simple: understand how things work under the hood — pointers, memory, and logic.

---

## Files Included

* `BubbleSortSLL.cpp` → Bubble Sort on Singly Linked List
* `CQ_arr.cpp` → Circular Queue (Array Implementation)
* `CQll.cpp` → Circular Queue (Linked List Implementation)
* `FloydWarshall_Implement.cpp` → All-pairs shortest path algorithm
* `LoopDetection.cpp` → Detect cycle in linked list
* `PalindromeInSLL.cpp` → Check if linked list is palindrome
* `Queue_LL.cpp` → Queue using Linked List
* `ReverseLL.cpp` → Reverse a singly linked list
* `SLL_Deletion_AnyPos.cpp` → Delete node at any position
* `SLL_Insertion_AnyPos.cpp` → Insert node at any position
* `Stack_Using_LL.cpp` → Stack using Linked List

---

## How to Run

Compile any file using:

```bash
g++ filename.cpp -o output
```

Run:

```bash
./output
```

On Windows:

```bash
output.exe
```

---

## Notes

* Most programs use a **minimal `main()`** for direct testing
* No unnecessary input handling — focused on logic
* Indexing (where applicable) is **0-based**
* Uses dynamic memory (`new` / `delete`)
